# Duder Blog

个人技术博客，使用 [Hugo](https://gohugo.io) 静态网站生成器构建，托管于 GitHub Pages。

## 关于

这是一个记录学习与思考的技术博客，内容涵盖：

- 医学成像技术
- 编程语言（C++、Python）
- 工具与效率
- 随想与感悟

## 技术栈

- **Hugo** — 静态网站生成
- **mogege 主题** — 基于 LeaveIt 的增强主题
- **Gitalk** — 评论系统
- **GitHub Pages** — 托管服务

## 特性

- KaTeX 数学公式渲染
- PlantUML 图表支持
- Bilibili 视频嵌入
- 深色/浅色主题切换
