AI时代——我回来了

Written by Duder with ♥ on May 10, 2026 in 随想, 692 words

前言

时隔许久，重新打开这个博客。之前一直在断断续续地写，但总有各种事情打断。终于在这个时间点，我想认真地把这个习惯捡起来——毕竟，在这个 AI 飞速发展的时代，有太多想法需要记录。

为什么是现在

之前一直在用 Gridea 写博客，但那套工具越来越不跟手。这次刚好趁着 AI 工具的东风，重新整理了一下博客的技术栈。用上了 Hugo 静态博客，配合 Claude Code 来辅助写作和调试，体验出乎意料地顺畅。（唯一的缺点可能就是费钱了吧。）

这次用 Claude Code 的感受

之前也用过各种 AI 助手，但 Claude Code 的体验确实不太一样：

  1. 代码理解能力强 — 不只是给答案，还能理解代码库的结构，给出符合项目风格的建议
  2. 上下文保持 — 在写 CLAUDE.md 的时候，它能记住整个项目的架构，给出的建议也更精准
  3. 可执行 — 能直接帮你运行命令、修改文件、提交代码，整个流程不需要我反复在终端和编辑器之间切换

特别是在整理博客的时候，Claude Code 帮我快速梳理了 Hugo 的目录结构，甚至还帮我生成了 CLAUDE.md 文件。这种"边聊边做"的体验，比之前用 ChatGPT 先问再复制粘贴要高效得多。（开始拉踩）

AI 时代的写作

我开始重新思考写作这件事。AI 不会替代我思考，但它可以承接很多重复性的工作：检查语法、整理排版、甚至帮忙想标题。真正的核心还是那些只有我能写出来的内容——经历、思考、独到的观察。

博客这个地方，我想保持它的手工感。

下一步

接下来我会陆续把之前积累的内容整理上来，同时也会尝试用 AI 工具来辅助写作和排版。如果这个流程跑顺了，后续会分享一些具体的经验。

这篇文章本身就是在 Claude Code 的辅助下完成的，整体思路和文字是我自己的（并不是），代码和流程上的问题交给 AI 处理。

声明：除了括号里的内容是我写的以外，其他都是Claude自己写的。
